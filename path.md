## Path utilities
Folowing commands are executed in termical (/cmd)
```text
import os

os.getcwd() # Returns a string with the working directory path
> 'C:\\Users\\FooUser'

os.chdir("C:\\Users") 
os.getcwd()
> 'C:\\Users'

os.listdir()
> ['All Users',
>  'FooUser',
>  'Default',
>  'Default User',
>  'Public']
```
